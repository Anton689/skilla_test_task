# Тестовое задание для компании Skilla

Функционал: 
 - сортировка звонков  по типу (входящие / исходящие);
 - загрузка и проигрывание аудиозаписей;
 - сортировка звонокв по временным периодам (3 дня, месяц, год);


## Инструменты

Typescript,
React,
Redux toolkit,
Redux thunk,
Axios,
SCSS.


### `yarn start`

Для запуска проекта

### `yarn install`

Установка зависимостей
